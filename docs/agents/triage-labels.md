# Triage Labels

技能使用五个标准分类角色。本文件将这些角色映射到本仓库实际使用的标签字符串。

| mattpocock/skills 中的标签 | 本仓库使用的标签 | 含义                       |
| -------------------------- | ---------------- | -------------------------- |
| `needs-triage`             | `needs-triage`   | 维护者需要评估此问题       |
| `needs-info`               | `needs-info`     | 等待提问者补充更多信息     |
| `ready-for-agent`          | `ready-for-agent`| 描述完整，AFK 代理可接手   |
| `ready-for-human`          | `ready-for-human`| 需要人工实现               |
| `wontfix`                  | `wontfix`        | 不予处理                   |

当技能提到某个角色（如"应用 AFK-ready 分类标签"），使用本表对应的标签字符串。

直接修改右列即可匹配你实际使用的标签词汇。
